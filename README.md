# AKS Primality Test & Complexity Theory
A simulation of the first deterministic, polynomial-time primality test.

### 🔬 Research Highlights
- **Polynomial Congruence**: Implements the $(x+a)^n \equiv x^n + a \pmod{n, x^r - 1}$ test logic.
- **Complexity Analysis**: Demonstrates the $O(\log^{12} n)$ (or optimized) complexity that proved Primality is in **P**.
- **Theory Focus**: A deep dive into Number Theory and Algebraic Complexity.

**Tech Stack:** Python, `SymPy`.
